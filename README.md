### Glucovibes — Predicción de Respuesta Glucémica Postprandial

Proyecto de data science completo para predecir el pico de glucosa y el tiempo hasta el pico tras una comida, basado en datos de CGM (Continuous Glucose Monitor) y logs de alimentación.

**Pipeline:**
1. `fase2_limpieza_local.py` — Limpieza de datos con justificación científica
2. `fase3_feature_engineering_local.py` — Ingeniería de características (glucosa basal, composición nutricional, etc.)
3. `fase4_clustering_modelado_local.py` — Clustering de perfiles glucémicos y modelado predictivo
4. `app.py` — Demo interactiva desplegada con Streamlit

**Tecnologías:** Python, Pandas, scikit-learn, Streamlit, clustering, regresión

### Federated Learning

Implementación de aprendizaje federado para entrenar modelos de ML de forma distribuida sin compartir datos sensibles.

- Módulos de formación: Welcome → Advanced Topics
- Implementación práctica con PyTorch y Flower framework
- Ejercicios en C para entornos embebidos
- Informe técnico completo

**Tecnologías:** PyTorch, Flower (flwr), Python, C
